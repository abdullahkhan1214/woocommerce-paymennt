== Description ==
This plugin can be used to enable 'Paymennt' payment method for the woocommerce shop websites. It enables iFrame or redirect mode of card payment.
You need to be registered with Paymennt (https://www.paymennt.com/) to functionally use this plugin.

To setup Paymennt plugin for card payments on woocommerce

Download the latest extension zip file from releases tab ( https://github.com/paymennt/woocommerce/releases/latest )
Upload the extension from your site Admin Panel Plugins -> Add New -> Upload Plugin,
Activate the Paymennt Card Payment plugin under Admin -> Plugins -> Installed Plugins
Manage Paymennt plugin settings from Plugins -> WooCommerce -> Settings -> Payments

== External Service Used ==
This plugin uses an external service: 'paymennt-php'  solely created by Paymennt itself. It serves as wrapper PHP sdk to access the Paymennt APIs.
The service's source code is accessible at https://github.com/paymennt/paymennt-php
